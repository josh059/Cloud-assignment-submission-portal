# Project Proposal

## Project Title

Cloud Assignment Submission Portal

## Background

Educational institutions need a secure and organized method of collecting assignments digitally.

## Problem Statement

Manual assignment submission methods are inefficient and prone to file loss and poor organization.

## Objectives

- Build a cloud-hosted application
- Enable secure assignment uploads
- Store files in Amazon S3
- Store metadata in DynamoDB

## Expected Outcomes

- Reduced paperwork
- Improved accessibility
- Secure storage
- Easy retrieval
